# samsara

Chromium extension for cycling through open tabs with user-configured shortcuts, similar to the macOS Cmd+Tab app switcher. Inspired by the native Arc/Dia browser tab switcher.

## Install locally

1. Open the extensions page in a Chromium browser.
2. Turn on **Developer mode**.
3. Click **Load unpacked** and select this folder.
4. Open the extension shortcuts page to change the shortcuts.

Default shortcuts:

- Next tab: `Ctrl+Shift+Comma` (`Command+Comma` on macOS)
- Previous tab: `Ctrl+Shift+Period` (`Command+Shift+Comma` on macOS)
